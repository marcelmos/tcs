1. "Aktualizacja:"
    Zawartość folderu "Projekt" przerzuć do folderu "ProjektTestowy" znajdujący się w folderze "htdocs" (C:\xampp\htdocs)
    Nadpisz pliki.

2.Uruchomienie XAMPP-a:
    Z katalogu xampp uruchom program "xampp-control.exe", w aplikacji wciśnij przycisk "Start" przy Module Apache oraz MySQL.
    Gdy wszystko jest dobrze uruchomione moduły powinny podświetlić się na zielono.

3.Uruchomienie witryny:
    W przeglądarce wpisz adres "localhost/ProjektTestowy", po poprawnym uruchomieniu XAMPP-a wszystko powinno działać.

4.Otwarcie bazy danych:
    W przeglądarce wpisz adres "localhost", z otwartej witryny z górnego menu wybierz "phpMyAdmin", po poprawnym uruchomieniu XAMPP-a wszystko powinno działać. Tam wystarczy z lewego pionowego menu wybrać baze danych "lokatorzytest".
